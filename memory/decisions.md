# Memoria Persistente: Decisiones (Decisions)

## Decisiones Estratégicas
- **Arquitectura de Documentación**: Implementación del estándar de "Memoria Fragmentada" (SmartScales 2 standard).
- **Consolidación de Niveles**: Clasificación neuronal estricta: ULTRA, PRO, AVANZADO, ESENCIAL.
- **Auto-Inteligencia**: Implementación de la "Silicon Elevation" para sintonización de hardware automática.
- **Pureza Visual**: Migración total a icons Lucide vectorizados (Emoji Purge).
- **Core de Inteligencia**: Uso mandatorio de `Transformers.js` con aceleración WebGPU local.
- **Estrategia PWA**: Enfoque *Offline-First* con Reactive Versioning y Update Toasts.
- **Aislamiento de Seguridad**: Configuración de cabeceras COOP/COEP en Vercel.

## Historial de Sesiones
- **2026-03-29**: Finalización del 100% de la documentación técnica del proyecto (9 módulos).
- **2026-04-02**: Consagración de la **Fase 10 (El Bautismo)**. Decisión de vincular la identidad a un ritual de naming para consolidar el "Identity Node".
- **2026-04-02**: Implementación de la **Bóveda de Inteligencia**. Decisión de persistir el hilo de chat (`chatHistory`) en el `userData` federado.
- **2026-04-11**: Implementación de **Crystal Edition (v3.6.0)**. Se resolvió la inestabilidad del motor de IA (Transformers.js v3), se integró biometría WebAuthn en el Diario y se desplegó un sistema de Auras personalizables con contraste adaptativo (APCA) para garantizar legibilidad premium.
- **2026-04-11**: Optimización del **Privacy Shield (FaceID)**. Decisión de implementar un flujo de registro automático (`navigator.credentials.create`) la primera vez para asegurar que Safari use el prompt de biometría directa (plataforma) en lugar del selector genérico de Passkeys en usos recurrentes.
- **2026-04-12**: Transformación de Arquitectura a **Vite**. Decisión de modernizar el flujo de construcción para permitir la gestión de secretos vía `.env`, integración nativa del SDK de Supabase y optimización de assets. Se desbloqueó temporalmente el `sw.js` para asegurar compatibilidad de caché con los nuevos artefactos de build.
- **2026-04-02**: Implementación de la **Sincronización Neuronal**. Decisión de desacoplar el aura del UX manual para vincularla a un "latido" sistémico que sintoniza con el reloj biológico del usuario y la carga cognitiva de Jade.
- **2026-04-02**: Implementación de la **Pasarela Biométrica-Atmosférica**. Decisión de otorgar a Jade la capacidad de "sentir" el estrés del usuario (HRV) y forzar atmósferas de calma autónomamente, priorizando el bienestar sobre la estética estática.
- **2026-04-02**: Consagración de la **Ingeniería Espacial**. Decisión de adoptar una topología de **Side-Rail** para iPad/Desktop y **Mobile-First** para iPhone, asegurando que Jade respire con naturalidad en cualquier factor de forma.
- **2026-04-02**: Implementación del **Blindaje de Estabilidad**. Decisión de activar el **Lazy Neural Activation** (Carga diferida de IA). Jade ahora solo carga modelos pesados bajo demanda (Chat/Portal), eliminando el riesgo de colapso de memoria (OOM) en dispositivos iOS.
- **2026-04-02**: Refuerzo del **Ciclo de Vida PWA**. Implementación de un guardián de recarga persistente (`sessionStorage`) para evitar bucles infinitos durante actualizaciones de Service Worker.
- **2026-04-12**: Auditoría de Integridad Técnica (Post-Vite). Se detectó y corrigió el uso de CDNs en `embedder.js` para asegurar el cumplimiento de COEP/COOP y funcionalidad offline. Se inyectó el header `x-device-id` en el cliente de Supabase para habilitar RLS. Se eliminaron logs sensibles y se validó el build de producción.
- **2026-04-12**: Estabilización de Dependencias. Migración definitiva a importaciones locales de Lucide y Transformers.js, eliminando dependencias de red externas críticas durante el tiempo de ejecución.
- **2026-04-12**: Refinamiento UI/UX ProMax. Se corrigió el bug crítico de apilamiento de vistas mediante el desacoplamiento de transiciones (absolute positioning en salida). Se elevó la estética a nivel premium con 30px de blur y bordes dinámicos. Se resolvió la inestabilidad de los listeners de Ajustes sincronizándolos con el ciclo de vida del renderizado asíncrono.
- **2026-04-12**: Integración Sensorial (Design Spells & Anime.js). Se formalizó el uso de micro-interacciones mágicas en el protocolo de agentes. Se implementó el motor `spells_engine.js` con el hechizo 'Magnetic Core' para elevar la presencia física de la IA, incrementando la tracción y el deleite del usuario.
- **2026-04-12**: Consolidación de la Simetría ProMax (v3.7.1). Se resolvieron las distorsiones elípticas en el *stats-grid* y se implementó una Arquitectura de Encabezado Global Inteligente. Esta decisión centraliza los controles de configuración y adapta dinámicamente el contexto visual (títulos/captions) según la vista activa, mejorando la navegación y eliminando el apilamiento de DOM (stacking bug).
- **2026-04-12**: Estabilización Técnica v3.7.1. Se eliminaron errores de runtime (`bgDownloader` duplicado), se corrigió el flujo de crypto para permitir el onboarding ininterrumpido y se migró la iconografía a importaciones ESM (`lucide`) para asegurar la integridad visual en el build de producción.
- **2026-04-12**: Hardening de Seguridad y UI (v3.7.2). Se implementó un protocolo de sal criptográfica aleatoria por usuario (`mqa_crypto_salt`), eliminando el riesgo de pre-computación de llaves. Se fortificó la ceremonia de onboarding con un flujo de recuperación verificado que valida la contraseña contra datos en la nube antes de permitir el acceso. Se desacopló la sincronización de atmósfera de la personalización de UI para eliminar riesgos de recursividad, centralizando la actualización en el flujo de guardado de ajustes.
