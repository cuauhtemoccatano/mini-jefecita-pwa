# 🌿 WellAlly Integration: Mini Jefecita Standards

El objeto debe ser el nodo central del bienestar de Jade.

## 1. El Puente Vital (HealthKit Bridge)
- **Protocolo de Atajos**: Los datos deben entrar vía URL Params para máxima velocidad.
- **Normalización**: Asegurar que los pasos se registren en números enteros y el HRV en milisegundos.

## 2. Conocimiento Preventivo
- Usar la filosofía de WellAlly: El bienestar es un equilibrio entre nutrición, movimiento y paz mental.
- La app debe priorizar el HRV (Variabilidad de Frecuencia Cardíaca) como el indicador maestro del estado del sistema nervioso de Jade.
